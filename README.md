# Space Atlas

Space Atlas made with Next.js project and Tailwind CSS.

## Quick Start

1. Install dependencies:
   ```sh
   pnpm install
   ```
2. Run the project:
   ```sh
   pnpm run dev
   ```

## Structure

- `app/` — pages and layout
- `components/` — UI components
- `hooks/` — custom hooks
- `lib/` — utilities
- `public/` — static files
- `styles/` — styles
